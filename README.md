# FlashCSS Eternity v7.0

**FlashCSS Eternity** هي مكتبة CSS و JavaScript تُسهل تصميم الواجهات والتفاعل معها بشكل مرن وسريع. تضم العديد من المزايا المتقدمة مثل UI Kit، دعم الأنماط الحسابية، دعم العمليات المنطقية، وأكثر!

## المميزات:
- **UI Kit**: مجموعة من المكونات الجاهزة لبناء واجهات المستخدم بسرعة وسهولة. تشمل عناصر مثل `navbar-glass`, `hero-3d`, `card-premium`, `btn-glow` وغيرها.
- **الدعم الكامل للـ CSS Shorthands**: دعم شامل لجميع الخصائص المركبة (مثل `p`، `m`، `w`)، مما يسهل الكتابة ويوفر الوقت.
- **العمليات الحسابية المتقدمة**: إمكانية استخدام العمليات الحسابية مثل `sin`, `cos`, `tan`, `min`, `max`, `clamp` في داخل الـ HTML.
- **دعم العمليات المنطقية**: استخدام العمليات المنطقية المتقدمة مثل `AND`, `OR`, `NOT` داخل الـ HTML.
- **إدارة السمات (Themes)**: إمكانية تغيير التصميم بسهولة باستخدام سمات مختلفة، مثل `light` و `dark`.
- **دعم التفاعل المتقدم**: مثل الحركات، الترجمة، التدوير، التكبير، التمدد، التغيير اللوني.
- **دعم التوافق التلقائي مع المتصفحات القديمة**: حقن تلقائي للـ polyfills ليعمل الموقع بشكل مثالي حتى على المتصفحات القديمة.
- **دعم الحركات الفيزيائية**: دعم تأثيرات 3D مثل `perspective`, `rotateX`, `rotateY`، مما يتيح إنشاء حركات ثلاثية الأبعاد واقعية.
- **دعم الانقسامات المعقدة**: بناء تصميمات معقدة باستخدام CSS Grid و Flexbox مع دعم واسع لخصائص مثل `justify`, `align`, `gap`, `z-index`.

## الاستخدام

يمكنك تحميل المكتبة عبر **CDN** باستخدام الرابط التالي:

```html
<script src="https://cdn.jsdelivr.net/gh/aimqst/flashcss@v1.0.0/flashcss.js"></script>
<div f-ui="navbar-glass">
    <span>الرابط 1</span>
    <span>الرابط 2</span>
</div>
<div f-ui="card-premium">
    <h2>عنوان البطاقة</h2>
    <p>محتوى البطاقة</p>
</div>
<div f-ui="hero-3d">
    <h1>مرحبا بك في الموقع</h1>
</div>
